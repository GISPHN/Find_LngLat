Find LngLat (説明を表示/非表示)

このページでは任意地点の緯度経度をコピペできます。
使い方：地図を動かしてクリックまたは 長押し（スマホ）/右クリック（PC）するとコメント登録や座標コピーが可能です。
地図スタイルは自由に切り替えられます。
また「地区踏査モード」で移動軌跡を記録し、コメントも入力できます。
利用地図：OpenStreetMap(©OpenStreetMap Contributors), osmfj_poi.json(K-Sakanoshita)
利用API：OpenStreetMap Nominatim API
作成者： @GISPHN
